# JetPack JoyRide

## Description

This is an arcade game similar to "Jetpack Joyride" developed completely using OpenGL and C++.

* Run the game:
	```console
	bar@foo:~/JetPackJoyRide/src$ make
	bar@foo:~/JetPackJoyRide/src$ ./Jetpack_Joyride
	```

## Bonus

* Enemy-4(Dragon)
* Sound
